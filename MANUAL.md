# 🏥 병원 홈페이지 운영/수정 매뉴얼

본 홈페이지는 **코드를 전혀 몰라도** `medical-content.json` 파일의 텍스트만 수정하면 사이트 내용이 실시간으로 변경되도록 설계되었습니다.

## 📂 데이터 파일 위치
`src/config/medical-content.json`

---

## ✏️ 수정 가이드

### 1. 병원 기본 정보 수정 (Clinic Info)
`clinic` 항목을 찾아 수정하세요.
```json
"clinic": {
  "name": "새로운 병원 이름", 
  "slogan": "메인 화면에 크게 나올 슬로건",
  "contact": {
    "phone": "02-1234-5678",
    "hours": "진료시간 수정"
  }
}
```

### 2. 원장님 프로필 수정 (Doctor Profile)
`doctor` 항목에서 약력과 철학을 수정할 수 있습니다.
- `image`: `public/images/` 폴더에 사진을 넣은 후, 이곳에는 `/images/파일명.jpg` 형태로 입력하세요.
- `history`: 약력은 콤마(,)로 구분하여 여러 줄을 추가할 수 있습니다.

### 3. 진료 과목 관리 (Services)
`services` 리스트에 새로운 항목을 복사하여 붙여넣으면 자동으로 카드가 추가됩니다.
```json
{
  "title": "New Service",
  "koreanTitle": "새로운 시술",
  "description": "시술 설명...",
  "duration": "30 mins",
  "recovery": "당일 회복"
}
```

---

## ⚠️ 주의사항 (의료법 준수)
1. **최상급 표현 주의**: '최고', '유일', 'No.1' 등의 표현은 의료법 위반 소지가 있으므로 자제해주세요.
2. **전후 사진**: 전후 사진을 게시할 때는 동일 조건 하에 촬영되었음을 명시해야 합니다. (별도 페이지 구성 시)

## 🚀 수정 후 반영
파일을 저장 (`Ctrl + S`) 하면 웹사이트에 즉시 반영됩니다. (개발 모드 시)
배포된 사이트의 경우, 수정 후 다시 배포(Deploy)가 필요합니다.
